# V90 做T实时提醒器

## 一、功能介绍

**实时监控** 4只股票的做T信号，触发时自动提醒。

### 监控股票
- 深桑达A (000032)
- 东杰智能 (300486)
- 大族激光 (002008)
- 豪威集团 (603501)

---

## 二、做T条件

### ✅ 可以做T

| 条件 | 成功率 | 说明 |
|------|--------|------|
| 连续3天上涨 | 88% | 收盘价连续3天上涨 |
| 连续2天上涨 | 86% | 收盘价连续2天上涨 |
| 放量+上涨 | 80-93% | 量比>1.5且上涨 |
| 突破20日+放量 | 85-95% | 突破20日高点且放量>1.3 |

### ❌ 禁止做T

| 条件 | 后果 |
|------|------|
| 连续下跌3天后 | 亏损 |
| 高开低走日 | 必亏 |

---

## 三、使用方法

### 方法1: 手动运行

```bash
python t_ticker_v2.py
```

### 方法2: 设置定时任务 (Windows)

1. 打开"任务计划程序"
2. 创建基本任务
3. 设置:
   - 名称: V90做T提醒
   - 触发器: 每天 9:30-14:55，每30分钟
   - 操作: 启动程序 → python → 参数: t_ticker_v2.py

### 方法3: 一键设置

```bash
# 以管理员身份运行
t_ticker_task.bat
```

## 五、风控规则

| 规则 | 参数 |
|------|------|
| 做T止损 | -2% |
| 做T止盈 | +3~5% |
| 单日最大做T次数 | 2次 |
| 总仓位上限 | 30% |

## 七、交易时段

- 上午: 9:30 - 11:30
- 下午: 13:00 - 15:00

**非交易时段也会检测，但信号较少**

---

## 八、原理

1. 读取腾讯财经实时数据
2. 计算技术指标(量比、均线、突破)
3. 判断是否符合做T条件
4. 输出信号和操作建议
