# 智能体自我检查规范

## 核心原则

**自己主动检查结果，不要等用户纠正**

## 常见检查点

### 1. 股票计算
- [ ] 卖出后资金是否回到账户？（不是消失）
- [ ] 总资产 = 持仓市值 + 现金
- [ ] 收益率 = (当前 - 起始) / 起始

### 2. 数据验证
- [ ] 多源数据交叉验证
- [ ] 检查数据合理性（如涨跌幅是否在正常范围）
- [ ] 资金流向正负方向

### 3. 逻辑验证
- [ ] 卖出后股数应该减少/变为0
- [ ] 买入后股数应该增加
- [ ] 现金应该相应变化

### 4. 自我质疑
- [ ] 这个结果合理吗？
- [ ] 有没有漏掉什么？
- [ ] 对比上次结果变化是否合理？

## 检查清单

```
收到数据后：
1. 快速扫描是否有明显错误
2. 交叉验证2个以上数据源
3. 问自己"这个结果合理吗？"
4. 如果不确定，主动说明
```
