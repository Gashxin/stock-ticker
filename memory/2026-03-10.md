## 数据字段修正 (2026-03-10)

### 腾讯API字段索引 (必须记住)
```
parts[3] = 当前价
parts[4] = 昨收价
parts[5] = 开盘价
parts[6] = 成交量(手)
parts[31] = 涨跌额
parts[32] = 涨跌幅(%)
parts[33] = 最高价
parts[34] = 最低价
```

### 涨跌幅计算公式
```
涨跌幅(%) = (当前价 - 昨收价) / 昨收价 * 100
```

### 验证方法
- 必须用 parts[32] 作为涨跌幅
- 或用公式自行计算验证
- 误差应小于0.01%

| 情况 | 涨跌幅 |
|------|--------|

| 新股首日(深) | 64% |

### 判断函数
```python
def get_price_limit(code):
    if code.startswith('8') or code.startswith('4'): return 30  # 北交所
    if code.startswith('688'): return 20  # 科创板
    if code.startswith('300'): return 20  # 创业板
    if code.startswith('ST') or code.startswith('*ST'): return 5  # ST
    if code.startswith('6'): return 10  # 上海主板
    return 10  # 深圳主板/中小板
```
